# jira

## requirement
- docker-compose: 17.09.0+

## How to run

- start jira

`docker-compose up`

- start jira daemon

`docker-compose up -d`

## How to hacker

`docker exec jira-srv java -jar /var/agent/atlassian-agent.jar  -p jira -m haxqer666@gmail.com -n haxqer666@gmail.com -o http://ip -s you-server-id-xxxx`
